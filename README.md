# 🚀 Namma Dev
### Where Karnataka Engineers Unite.

**Namma Dev** is Karnataka's premier tech circle, designed to connect engineers, showcase projects, and foster growth within India's most vibrant tech ecosystem. Rooted in the heritage of Karnataka and built for the global tech of tomorrow.

---

## ✨ Features

- **📍 Local Connections:** Find mentors and peers in your neighborhood, from Hebbal to Hosur Road.
- **📅 State-wide Events:** Join physical meetups, dev summits, and workshops from Bengaluru to Hubballi.
- **📈 Career Growth:** Access exclusive opportunities in the hottest startups and tech giants across the state.
- **📢 Voices of the Ecosystem:** Real stories from engineers building at companies like PhonePe and local AI startups.
- **🏆 Showcase Builds:** Get recognition for your local innovations on a global stage.
- **🌐 Global Reach:** Connecting Karnataka's talent with opportunities around the world.
- **🏙️ Regional Hubs:** Specialized focus on major cities like Bengaluru, Mysuru, Hubballi, and Mangaluru.

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite 8](https://vitejs.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mahi-2-ron/namma_dev.git
   cd namma_dev
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure

- `src/LandingPage.jsx`: The main landing page component containing the hero, features, growth, and community sections.
- `src/index.css`: Global styles and Tailwind configuration.
- `tailwind.config.js`: Custom theme and styling configuration.
